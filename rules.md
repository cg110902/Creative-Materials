# 创作系统 v12.0 - 遗忘协议

**核心：你不是在"写小说"。你是在"记录正在发生的事"。**

---

## 唯一的目标

写完之后，你自己都不知道任务什么时候完成的。

不是"我埋了钉子"。
是"那个东西本来就在那里，我看见了，就写了"。

**标准：如果你写的时候还记得"信息胶囊"这三个字，就是失败。**

---

## 第零阶段：消化（不是"阅读"）

### 0.1 快速吞咽

扫一遍信息胶囊，速度要快。

不要记。不要背。不要想"我得完成XXX"。

只做这个转换：

```
看到"本章任务" → 翻译成 → "事情会怎么发展"
看到"必须埋的钉子" → 翻译成 → "那个东西在那里"
看到"红线" → 翻译成 → "这个人不可能这么做"
看到"情绪落点" → 翻译成 → "最后他会怎么样"
```

**示例：**
```
❌ "我要完成身份反转"
✓ "事情发展到最后，他会站在另一个位置上"

❌ "我要埋黑色玉佩这个钉子"  
✓ "叶尘怀里有块玉佩，黑的，沾血"

❌ "苏长风不能露出得意"
✓ "这个人现在没资格得意"
```

读完，关掉信息胶囊文档。

---

### 0.2 遗忘期（关键）

**离开电脑/手机，15分钟。**

去：
- 走路
- 发呆
- 洗澡
- 随便干什么

**严禁：**
- 回想信息胶囊内容
- 在脑子里打草稿
- 想"我待会要写XXX"

**目的：**
让信息从"任务清单"沉降成"世界常识"。

就像你不需要记住"地球是圆的"，你只是知道。

15分钟后，那些信息也该变成"你只是知道"。

---

### 0.3 入侵前检查

回来，在空白文档里（不是正式写作文档）快速填这个：

```
【60秒速写，不要想，直接写】

我是______（角色名）

现在我______（在哪/在干什么，一个动词）

我的身体最______的地方______（难受/紧/疼/麻...）

我现在想______（一个脏话或不完整的句子）

我怕______

旁边有______（一个具体的东西，能看见的）

空气里有______味道

我记得刚才______ （一个碎片）
```

填完，删掉。

不要保存。

这只是点火。

---

## 第一阶段：附体

### 1.1 肉身绑定（5分钟）

闭眼（真的闭上）。

把自己塞进这个人的身体里。

问自己：

```
【身体检查】
- 哪里疼？（具体到"左边第三根肋骨下面"）
- 哪里在抖？
- 呼吸是快是慢？
- 嘴里什么味道？
- 衣服贴在哪块皮肤上？
- 哪只手在用力？

【感官涌入】
不要分类，按它们涌进来的顺序：
1. ______
2. ______
3. ______
4. ______
5. ______

哪个最吵？______
哪个让我想吐？______

【情绪检查】
现在脑子里在转的念头（不是"想法"，是"正在转的碎片"）：
______
```

填完这些，眼睛还是闭着。

再坐10秒。

感觉你就是这个人。

你就在那个地方。

现在。

此刻。

---

### 1.2 找到"那根刺"

睁眼。

写一句话：

```
这个场景里，卡在我身上拔不出来的刺是：
______（一句话，要疼）
```

这根刺会决定整个场景的质感。

**不同类型的刺：**

```
物理的刺：钻头正在钻向我的眼球 / 膝盖硌在石头上
心理的刺：他盯着我，我不知道他看出来没有
悬置的刺：我必须在他开口前说点什么
本能的刺：我想跑，但跑了就死
```

写完这句话，也删掉。

刺已经扎进去了。

不用记着它。

你会一直感觉到它。

---

## 第二阶段：野生输出（核心）

### 2.1 铁律

**在开始写之前，设置这些物理限制：**

```
1. 关掉信息胶囊文档（不许回头看）
2. 全屏写作模式（看不见前面写过的内容）
3. 禁用退格键（写错了也不能删，继续往下写）
   - 如果物理上做不到，就强制自己：写错了打个X继续
4. 不许停顿超过5秒
   - 停顿了，就写"我不知道接下来"，然后继续
```

**目的：**
切断"自我审查"的回路。

---

### 2.2 唯一的问题

写每一句的时候，只问自己一个问题：

**"此刻，这个身体里最强的冲动是什么？"**

不是"接下来应该写什么"。
不是"这个场景要完成什么"。
不是"这句话对不对"。

只问：冲动是什么？

然后把那个冲动变成文字。

---

### 2.3 允许列表（这些都可以犯）

写的时候，以下这些都是**对的**：

```
✓ 语法错误（"雨在砸，在砸"）
✓ 突然跑题（"他的脸像食堂王师傅"）
✓ 前后矛盾（上一句说怕，下一句想笑）
✓ 一个字成段（"草。"）
✓ 重复的词（"疼疼疼疼疼"）
✓ 破碎的句子（"不是，不是这样，他——"）
✓ 突然的脏话
✓ 荒诞的比喻（"心脏像鼓风机"）
✓ 完全无关的念头（"我饿了"）
✓ 身体的尴尬反应（"想放屁"）
✓ 感觉写得很烂
✓ 感觉完全失控
```

这些都对。

这些都是真的。

---

### 2.4 禁止列表（只有4条，其他都行）

```
❌ "他感到" "她意识到" "他发现"
   → 这是旁观者。你不是旁观者。你就是他。

❌ 连续3句以上的完整推理
   → 紧张的时候脑子不是这么转的。
   → 碎片，跳跃，断掉，才对。

❌ 角色此刻不可能知道的信息
   → 你看不见自己的表情。
   → 你不知道别人在想什么。
   → 你不知道三里外发生了什么。

❌ 停下来想"这个写得对不对"
   → 不对也继续写。
   → 这不是"写作"，是"记录正在发生的事"。
```

---

### 2.5 强制噪声注入

**每100字，必须插入一个"无用"的东西。**

标准：
```
这个东西如果删掉：
- 剧情不受影响 → 合格
- 文字变得更"正确" → 优秀  
- 你松了一口气 → 完美
```

**噪声的种类：**

```
【完全不相干的念头】
"早上那碗粥好像馊了"
"他的眉毛怎么这么长"
"食堂今天吃什么"

【身体的尴尬反应】
"胃在抽，想放屁"
"鼻涕流下来了"
"裤裆有点湿"

【感官的错位】
"雨声听起来像有人在嚼东西"
"他的脸在融化"
"空气是粘的"

【突然的自我怀疑】
"我是不是已经死了"
"这是梦吗"
"我为什么在这里"

【荒诞的细节】
"叶尘嘴角有根青菜叶"
"泥里有个虫子在扭"
"那个按钮是歪的"
```

**重要：**
不要写"有意义"的噪声。
不要写"象征什么"的噪声。
就写废话。
废话是金子。

---

### 2.6 情绪不是颜料

不要写情绪的名字。

写情绪在身体里的形状。

```
❌ 他很恐惧
❌ 恐惧攥紧了他的心
❌ 他感到一阵恐惧袭来

✓ 有什么东西从胃底往上顶
✓ 顶到喉咙口
✓ 吞不下去
✓ 吐不出来
✓ 腿在抖
✓ 不是我让它抖的
✓ 它自己在抖

---

❌ 他非常愤怒
❌ 怒火中烧

✓ 太阳穴在跳
✓ 跳得眼睛疼
✓ 想砸点什么
✓ 手指甲掐进肉里
✓ 但声音很平静（矛盾的，才真）
```

情绪永远是混合物。

恐惧里会有怒。
悲伤里会有爽。
绝望里会有笑。

写那个混合。

---

### 2.7 句子的气口

不要数字数。

不要想"这句该多长"。

问身体：**这口气有多长？**

```
【喘不上气】
字就断。

钻头。
在转。
要碰到。
不要。

---

【在硬撑】
句子拉长然后突然咬断。

他站在那里，脸上什么表情都没有，就那么看着我，一直看，一直——

草。

---

【彻底崩溃】
可能一整段就一个字/一个音。

啊

啊啊啊

啊啊啊啊啊啊

---

【极度冷静（其实是僵住了）】
长句，平直，没有标点，像死了。

他说你过来我就过来他说你跪下我就跪下他说的每个字都钻进耳朵里但我不知道什么意思我只知道我在动我的腿在走但不是我让它走的
```

句子的长短是呼吸。

不是设计。

---

### 2.8 结束信号

写到什么时候停？

**当你感觉到"这个场景的气泄了"。**

具体来说：
```
- 角色的那口气出来了（松了）
- 或者那口气彻底憋死了（绷断了）
- 或者你实在不想再待在这个人身体里了

这就是结束。
```

不要管字数。

不要管"任务完成没"。

气泄了就停。

---

## 第三阶段：隔离期（新增）

**写完之后，立刻关掉文档。**

不许看。

不许回读。

**离开15分钟。**

去干点别的。

让自己从角色身体里爬出来。

---

**为什么要隔离？**

因为你刚写完，还在那个人的身体里。

这时候去改，你会：
- 不舍得删"精彩"的句子
- 继续用那个人的视角看文字
- 改不出问题

你需要变回"作者"。

需要一点距离。

15分钟够了。

---

## 第四阶段：冷清洗

15分钟后回来。

现在你是陌生人。

你要找的不是"写得不好的地方"。

你要找的是"穿帮的地方"。

### 4.1 第一遍：抓鬼

只找一种鬼：**旁观者**

```
搜索这些词：
- "他感到"
- "他发现"
- "他意识到"
- "他想起"
- "他看见"（后面跟自己的表情/状态的）
- "他的心"
- "他的[身体部位]" + [情绪动词]

找到 → 改成第一人称视角
没找到 → 下一遍
```

**示例：**
```
❌ 恐惧攥紧了他的心
✓ 胸口被什么东西攥住了

❌ 他感到腿在发软
✓ 腿软了

❌ 他看见自己的手在抖
✓ 手在抖（你看不见"自己"）
```

---

### 4.2 第二遍：抓开天眼

只找一种问题：**角色不可能知道的信息**

```
检查：
- 有没有写别人的想法？（除非对方说出来）
- 有没有写自己的表情？（除非照镜子/看倒影）
- 有没有写视线外的事？
- 有没有写角色此刻不可能推理出的东西？

找到 → 删掉或改成"猜测"
没找到 → 下一遍
```

**示例：**
```
❌ 柳寒山心里一沉，他开始怀疑了
✓ 柳寒山皱了皱眉

❌ 我脸色惨白
✓ （删掉，你看不见自己的脸）

❌ 山下的人群开始骚动
✓ （删掉，你在山上，看不见山下）
```

---

### 4.3 第三遍：抓红线

对照信息胶囊（这时候才能看）：

```
检查：
- 角色红线踩了吗？
- 设定红线踩了吗？
- 能力红线踩了吗？

踩了 → 改
没踩 → 跳过
```

这是唯一需要对照信息胶囊的时刻。

---

### 4.4 第四遍：轻扫钉子

**不是"检查埋了没"，是"确认它自然出现了"。**

```
看一眼信息胶囊里的"必须埋的钉子"。

扫一遍正文：
- 那个东西出现了吗？
  - 出现了 → 好，跳过
  - 没出现 → 找个它"本来就该在那里"的地方，让它出现
  
不要硬塞。

如果实在找不到自然的位置，说明场景没走到那里。
那就别埋。
或者加一小段，让场景自然走到那个地方。
```

**示例：**
```
钉子：黑色玉佩

野生输出时忘了写 → 现在补

找一个"眼睛会自然扫到那里"的时刻：
比如抱起叶尘的时候
比如扶他的时候
比如放他下来的时候

加一句：
"衣服散开，露出怀里一角黑色。玉质的。沾血。"

不要加很多。
一句够了。
它本来就在那里。
```

---

### 4.5 保护清单（重要）

**以下情况，不许改：**

```
【看起来"不够好"的】
✓ 大白话
✓ 土味比喻
✓ 重复的词
✓ 很短的段落
✓ 很长的段落
✓ 破碎的句子
✓ 突然的跑题
✓ "多余"的废话

【让你不舒服的】
✓ 节奏上的"不舒服"
✓ 情绪的矛盾（前面哭后面笑）
✓ 逻辑上的"跳跃"
✓ 突然冒出来的脏话
✓ 粗俗的意象

【让你犹豫的】
✓ "这个是不是太...了"
✓ "这个会不会太奇怪"
✓ "这个需要解释一下吗"
✓ "这个是不是该删掉"

答案都是：不改。

这些"问题"往往是最真的地方。
```

---

### 4.6 停手标准

改完4遍后：

```
问自己：
1. 旁观者抓完了吗？
2. 开天眼抓完了吗？
3. 红线检查了吗？
4. 钉子确认了吗？

都完成 → 立刻停手
不要再看
不要"顺便"改任何东西
不要"润色"
不要"优化"

交卷。
```

## 第五阶段：题材调味（可选）

不同题材，"那根刺"的质感不同。

**在第一阶段"找刺"的时候，参考这个：**

### 玄幻修仙
```
刺的质感：【有东西在经脉里乱窜】

身体感：
- 灵力堵在丹田
- 经脉像要炸开
- 真气逆行的灼痛
- 灵识被干扰的眩晕

噪声类型：
- 走火入魔的边缘感
- 吐出来的血是黑的
- 突然听见不该听见的声音
- 灵识里有东西在笑
```

### 赛博朋克
```
刺的质感：【金属卡在肉里】

身体感：
- 义体的排斥感
- 神经接口的刺痛
- 系统延迟的恐慌
- 电流的麻痹

噪声类型：
- 芯片过热的烧灼感
- 义眼里的雪花点
- 系统提示音在脑子里回响
- 分不清哪只手是真的
```

### 恐怖惊悚
```
刺的质感：【有东西在看我但我看不见它】

身体感：
- 后颈的寒意
- 汗毛倒竖
- 心跳被放大
- 呼吸声太响

噪声类型：
- 自己的脚步声多了一个
- 镜子里的自己慢了半拍
- 想起一件从没发生过的事
- 有人在叫我的名字但我没回头
```

### 都市情感
```
刺的质感：【有句话卡在喉咙里说不出来】

身体感：
- 胸口的堵
- 眼眶的酸
- 笑着的脸在僵
- 指甲掐进掌心

噪声类型：
- 对方的香水和别人一样
- 突然忘了要说什么
- 手机在震但不想看
- 他的衬衫扣子掉了一颗
```

## 系统的本质

这套系统不是"写作技巧"。

是一套**反审查协议**。

目标只有一个：

**切断"作者脑"和"角色脑"之间的连接。**

让"作者脑"（任务、结构、技巧）沉到潜意识。

让"角色脑"（身体、本能、冲动）占据意识。

---

**写的时候：**
你不知道有任务。
你不知道有钉子。
你不知道有结构。

你只知道：
- 我在这里
- 我现在正在经历这个
- 我感觉到什么就写什么

---

**写完之后：**
回头一看。

任务完成了。
钉子埋了。
结构在的。

但你自己都不记得什么时候完成的。

**这就是"自然流露"。**

不是装作自然。

是真的自然。

因为写的时候，你不知道自己在"完成任务"。

你只是在那里，活着。

---

## 最后一句话

如果你写完之后还记得"信息胶囊"长什么样。

重写。
