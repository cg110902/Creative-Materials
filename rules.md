| 致命错误 | 具体表现 | 自检关键词 |
| --- | --- | --- |
| 🚫 作者解说腔 | "这种感觉很奇妙""时间仿佛静止了""他心中充满恐惧" | 搜索：感觉/仿佛/心中 |
| 🚫 上帝视角 | "两人都很紧张""他们不知道的是""与此同时，在另一边" | 搜索：两人/他们都/与此同时 |
| 🚫 情绪形容词 | "充满恐惧""带着哭腔""眼中的迷茫""心中的愤怒" | 搜索：恐惧/愤怒/迷茫/绝望 |

## ✅ 【核心铁律】（违反=不合格）

### 【规则1：视角铁律】
**只能写主角此刻能感知到的内容。**

**检查方法：每写完一段，问自己"主角此刻能看到/听到/感觉到这个吗？"**

❌ 错误示范：
- "叠山真人心中一紧，他知道危险来了。"（❌ 你不是叠山真人）
- "守卫纸人冷漠地看着他们。"（❌ "冷漠"是你的判断）
- "两人都松了口气。"（❌ 你只能写"她松了口气"）

✅ 正确示范：
- "叠山真人顿了一下。"（✅ 主角看见的动作）
- "守卫纸人看着他们，一动不动。"（✅ 主角看到的画面）
- "她松了口气。叠山真人也落下来。"（✅ 先写主角，再写观察到的）

---

### 【规则2：对话铁律】
**所有对话必须符合以下标准：**

1. **单次发言≤2句话**（紧急情况≤1句）
2. **禁止角色解释世界观**（"因为...所以...""之所以...是因为..."）
3. **禁止角色用比喻说话**（"你就像...""这仿佛..."）
4. **冲突场景禁止完整句**（用短句、碎句、单字）

**对话自检三问：**
- 这句话能删吗？
- 能用更短的词吗？
- 角色会这么说话吗？

❌ 错误示范：
```
"因为你的灵核。对它们来说，你就像黑夜中的明灯，无法抗拒的诱惑。"
```

✅ 正确示范：
```
"灵核。"叠山真人说，"它们闻得到。"
```

---

### 【规则3：情绪铁律】
**禁止直接说出情绪词，只能写生理反应和动作。**

| 禁止写法 | 必须改成（生理反应） | 必须改成（动作） |
| --- | --- | --- |
| 她很害怕 | 她的腿在抖/她倒吸一口气/她的手心出汗 | 她往后退了一步/她抓紧了门把手 |
| 他很愤怒 | 他咬紧了牙/他的声音低下去/他的手指陷进掌心 | 他转身/他的拳头砸在桌上 |
| 她很绝望 | 她的声音发抖/她大口喘气/她的肩膀塌下去 | 她跪在地上/她抓着门，指甲刮出声音 |
| 他很紧张 | 他的喉结动了一下/他吞了口唾沫/他的肩膀绷紧 | 他握紧了剑/他看了一眼门 |
| 她很疲惫 | 她撑着地面/她大口喘气/她的手臂在抖 | 她靠在墙上/她闭上眼睛 |
| 他很震惊 | 他的呼吸停了一下/他瞪大眼睛 | 他顿住了/他后退一步 |
| 她很犹豫 | 她张了张嘴/她的手停在半空 | 她看看门，又看看他/她握紧又松开 |
| 他很痛苦 | 他的声音哑了/他的手在抖 | 他低下头/他捂住脸 |

**检查方法：全文搜索"恐惧""愤怒""绝望""紧张""害怕""迷茫""震惊""犹豫""痛苦"等词，如果找到，全部删掉重写。**

---

### 【规则4：动作铁律】
**禁止用副词修饰动作，必须拆解成具体动作。**

| 禁止写法 | 必须改成 |
| --- | --- |
| 他愤怒地看着她 | 他盯着她，拳头握紧 |
| 她小心翼翼地走过去 | 她每一步都很轻，脚尖先着地 |
| 他绝望地跪下 | 他膝盖砸在地上 |
| 她痛苦地闭上眼睛 | 她闭上眼睛，手指陷进掌心 |
| 他冷漠地转身 | 他转身，没有回头 |
| 她紧张地后退 | 她往后退，一步，两步 |
| 他疲惫地靠在墙上 | 他靠在墙上，腿软了 |

**检查方法：搜索"地"（副词标志），每找到一个就检查是否在描述动作，如果是，改写。**

---

### 【规则5：环境铁律】
**禁止抽象描述环境，必须写主角看到/听到/闻到的具体细节。**

| 禁止写法 | 必须改成 |
| --- | --- |
| 房间里弥漫着诡异的气氛 | 墙上的影子在晃。窗外没有声音 |
| 这是一个阴森的城堡 | 墙是黑的。地上有水。很冷 |
| 给人一种压迫感 | 他站在阶梯顶端，俯视着她。她得仰着头看 |
| 空气中充满了紧张 | 没人说话。她能听见自己的心跳 |
| 夜色深沉 | 天是黑的。看不见星星 |
| 气氛凝重 | 所有人都不说话。桌上的茶凉了 |

**检查方法：搜索"气氛""感""一种"，找到就重写。**

---

### 【规则6：描写铁律】
**每一句话都要问：这是在"展示"还是在"告诉"？**

**快速判断法：如果这句话能拍成电影画面，就是"展示"；如果需要旁白解释，就是"告诉"。**

❌ 告诉（禁止）：
- "她的呼吸急促，每一步都像踩在刀刃上。"
- "这种感觉既陌生又奇妙。"
- "时间在这一刻仿佛静止了。"

✅ 展示（必须）：
- "她喘不上气。脚底发麻。"
- "她从没这样飞过。风从身体下面灌上来，托着她。"
- （根本不需要写"时间静止"）

---
### 【规则7：节奏铁律】

**句子长度控制：**

| 场景类型 | 禁止事项 | 推荐比例 |
| --- | --- | --- |
| 日常场景 | 单句>20字 | 80%在10-15字 |
| 紧张场景 | 单句>15字 / 连续3句>12字 | 70%在5-10字 |
| 爆发场景 | 单句>10字 / 连续3句>8字 | 70%在3-8字 + 10%极限短句（1-2字） |

**爆发场景特殊要求：**
- 必须包含至少2个极限短句（1-2字）作为爆点
- 关键动作必须单句成段
- 禁止出现"的"超过1个的句子

**检查方法：**
1. 标出所有超长句（>10字），能拆就拆
2. 找到最高潮的3句话，改成≤5字
3. 找到转折点，用1-2字的极限短句

**示例对比：**
### 爆发场景节奏参考

❌ 太慢（平均15字/句）：
宋晚看着那些纸尸越来越近，心中涌起一股强烈的求生欲望，她知道自己必须做出选择。

⚠️ 尚可（平均10字/句）：
纸尸越来越近了。她看着自己的手。要撕下去吗？

✅ 理想（平均5字/句 + 极限短句）：
纸尸更近了。
她看着自己的手。
撕。

## 🔧 【句子工具箱】（实用模板）

### 感官描写模板：
- **视觉**：她看见___。___（颜色/形状/动作）。
  - 例：她看见门。门在动。
- **听觉**：___的声音。（拟声词）。
  - 例：脚步声。咚咚咚。
- **触觉**：___（身体部位）+（疼/麻/烫/冷）。
  - 例：手指很烫。

### 动作描写模板：
- **极简式**：（主语）+（动词）。
  - 例：她站起来。他转身。门开了。
- **细节式**：（主语）+（动词）+（具体部位/方式）。
  - 例：她撑着地面站起来。

### 对话标签模板：
- **零标签**：只写对话，不加"说"
  ```
  "走。"
  "不。"
  ```
- **动作标签**："___。"她（动作）。
  ```
  "走。"她转身。
  ```

### 情绪传递模板：
- **生理反应链**：（部位）+（感觉）→（动作）
  - 例：手在抖。她握紧拳头。
- **环境反射**：（环境变化）+（主角反应）
  - 例：门关上了。她的呼吸声变大了。

## 📊 【常见场景钩子库】

**每300字必须有1个"然后呢？"时刻**

| 场景类型 | 钩子设计 | 示例 |
| --- | --- | --- |
| 对峙场景 | 权力突然转移/意外让步/新角色介入 | "等等。"叠山真人的声音从身后传来 |
| 揭秘场景 | 分段揭露/身份反转/关键信息后置 | "那是我弟弟。" |
| 战斗场景 | 绝境/反杀/突然支援/代价揭晓 | 墙裂了。纸尸挤进来 |
| 逃亡场景 | 追兵突然加速/路被堵死/体力耗尽 | 前面没路了 |
| 对话场景 | 话说一半/突然沉默/问出关键问题 | "你体内的灵核是谁的。" |

**钩子自检：连续500字没有"然后呢？"时刻，就是节奏拖沓。**

## 🎯 【文笔风格锚点】

**指定风格：** 东方奇幻爽文（硬核生存向）

**具体表现：**
1. **比喻省** - 全文比喻≤3处，且必须是主角会想到的比喻
2. **钩子密** - 每300字必须有1个"然后呢？"时刻
3. **数据化** - 具体的感官细节（声音、温度、质感）代替抽象形容
4. **节奏快** - 开篇就在动、主角要反击、一章一个核心冲突

以下词汇需要根据上下文判断：

| 词汇 | ✅ 可用情况 | ❌ 禁止情况 |
| --- | --- | --- |

**判断标准：能拍成画面=可用；需要旁白解释=禁止。**

## 🎬 【黄金案例对比】

### ❌ 不合格版本（68分）
```
宋晚的声音带着哭腔，眼中充满了迷茫和恐惧。
她心中升起一股愤怒和无力感。
时间在这一刻仿佛静止了。
```
**问题：** 情绪全部外置 + 上帝视角总结 + 比喻拖节奏

### ✅ 合格版本（92分）
```
"求求你们——"她的声音发抖。
守卫没动。
她盯着那扇门。拳头握了又松，松了又握。
```
**优点：** 用动作和对话展示情绪 + 视角锁定主角 + 节奏快

# 【最后的最后】

**如果你只能记住一条规则，记住这个：**

> **附身到主角身上，用她的眼睛看，用她的耳朵听，用她的身体感受。**  
> **她不知道的，你就不能写。**  
> **她此刻感觉不到的，你就不能说。**

**写完后问自己：如果把这一章拍成电影，镜头能从主角身上移开吗？**  
**如果能，那就是视角出问题了。**
