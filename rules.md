# 【第一部分：创作前看】（固定不变）

## 💀 致命错误（这4条必须避免）

1. **上帝视角** - 不写主角看不到/不知道的事
   - ❌ "两人都很紧张" 
   - ✅ "她握紧拳头。他也在。"

2. **情绪外置** - 不直接说情绪词，用身体说话
   - ❌ "她很害怕"
   - ✅ "她的腿在抖"

3. **角色讲课** - 对话要像人话，不能变成说明书
   - ❌ "因为你的灵核，对它们来说就像黑夜中的明灯"
   - ✅ "灵核。它们闻得到。"

4. **空洞比喻** - 少用"仿佛""像""宛如"，多写实感
   - ❌ "时间仿佛静止了"
   - ✅ 直接删掉这句

---

## ✨ 核心原则（记住这3条就够了）

### 1. 附身原则
**"我就是主角，用她的眼睛看世界"**

写的时候问自己：主角此刻能看到/听到/感觉到这个吗？
- 能 → 写
- 不能 → 删

### 2. 展示原则  
**"能拍成画面吗？"**

- 能拍成画面 = 好的
- 需要旁白解释 = 不好

### 3. 呼吸原则
**"读起来顺吗？"**

写完后大声读一遍：
- 一口气读完，不卡壳 = 节奏对了
- 像背电报 = 句子太短太碎
- 像念文件 = 句子太长太拖

---

## 🎯 番茄爽文特征（风格指南）

1. **节奏活** - 长短句搭配，别全是短句或全是长句
   - 主力句：10-18字（60%）
   - 短句点缀：5-9字（30%）  
   - 爆点极短句：1-4字（10%）

2. **对话脆** - 一次说话≤2句，紧急时≤1句

3. **钩子密** - 每300-500字要有1个"然后呢？"的时刻

4. **开篇快** - 直接进入动作，不磨蹭

---

## 🔧 实用工具箱

### 当你不知道怎么写情绪时：

| 想写的情绪 | 改成身体反应 | 或者改成动作 |
|---|---|---|
| 害怕 | 腿在抖/手心出汗/倒吸一口气 | 往后退/抓紧门把手 |
| 愤怒 | 咬紧牙/手指陷进掌心 | 转身/拳头砸在桌上 |
| 震惊 | 呼吸停了一下/瞪大眼睛 | 顿住了/后退一步 |
| 犹豫 | 张了张嘴/手停在半空 | 看看这，又看看那 |

### 当对话太长时：

**长对话拆解法：**
```
❌ "我知道你在想什么，但是这件事没那么简单，我们必须小心。"

✅ "我知道你在想什么。"他顿了顿，"我们必须小心。"
```

### 当节奏拖沓时：

**快速自检：**
- 随机数100字，应该有5-8个句子
- 连续超过4个短句（≤5字）→ 合并
- 连续出现长句（>20字）→ 拆分

## 📖 黄金案例（感受节奏）

### 日常场景（舒缓）
```
她推开门，冷风灌进来。（10字）
屋子里很暗，只有窗边漏进一点光。（15字）
师父不在。（4字）
桌上的茶凉了，茶叶沉在杯底。（14字）
```

### 紧张场景（加速）
```
脚步声。（3字）
越来越近。（4字）
她盯着门，手按在墙上。（10字）
门把手在动。（5字）
```

### 爆发场景（冲击）
```
纸尸的手抓住她的脚踝。（11字）
冰凉。（2字）
她低头。（3字）
手指陷进皮肤里，渗出血来。（12字）
撕。（1字）
```
