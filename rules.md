# 【第一部分：创作前看】（固定不变）

## ⚠️ 【致命错误清单】⚠️
**以下任何一条出现，本章直接判定为不合格：**

| 致命错误 | 具体表现 | 自检关键词 |
| --- | --- | --- |
| 🚫 作者解说腔 | "这种感觉很奇妙""时间仿佛静止了""他心中充满恐惧" | 搜索：感觉/仿佛/心中 |
| 🚫 上帝视角 | "两人都很紧张""他们不知道的是""与此同时，在另一边" | 搜索：两人/他们都/与此同时 |
| 🚫 情绪形容词 | "充满恐惧""带着哭腔""眼中的迷茫""心中的愤怒" | 搜索：恐惧/愤怒/迷茫/绝望 |
| 🚫 过度比喻 | "像刀刃一样""仿佛黑夜中的明灯""如同萤火虫" | 搜索：像/仿佛/如同/宛如 |
| 🚫 角色讲课 | 角色说话超过2句连续对话/用比喻解释/说"因为...所以..." | 数对话句数/搜索：因为 |
| 🚫 总结式心理 | "她意识到""她明白了""她感觉到一种前所未有的" | 搜索：意识到/明白了/感觉到 |

## ⚠️ 【Claude最爱犯的10大错误】⚠️
**看到这些立刻警觉！这是Claude的本能坏习惯：**

1. ❌ "她感到一种前所未有的恐惧" → ✅ "她的腿在抖"
2. ❌ "两人对视一眼，都看出了对方眼中的决心" → ✅ "她看着他。他点了点头"
3. ❌ "空气仿佛凝固了" → ✅ "没人说话"
4. ❌ "时间在这一刻静止了" → ✅ 直接删掉这句
5. ❌ "他的声音中带着一丝不易察觉的颤抖" → ✅ "他的声音在发抖"
6. ❌ "她小心翼翼地推开门" → ✅ "她推开门，很轻，没有声音"
7. ❌ "内心充满了复杂的情绪" → ✅ "她握紧拳头，又松开"
8. ❌ "像是被什么东西狠狠击中" → ✅ "她顿住了"
9. ❌ "这让她想起了..." → ✅ 直接写回忆画面
10. ❌ "不知过了多久" → ✅ 要么写具体时间，要么不写

## ✅ 【核心铁律】（违反=不合格）

### 【规则1：视角铁律】
**只能写主角此刻能感知到的内容。**

**检查方法：每写完一段，问自己"主角此刻能看到/听到/感觉到这个吗？"**

❌ 错误示范：
- "叠山真人心中一紧，他知道危险来了。"（❌ 你不是叠山真人）
- "守卫纸人冷漠地看着他们。"（❌ "冷漠"是你的判断）
- "两人都松了口气。"（❌ 你只能写"她松了口气"）

✅ 正确示范：
- "叠山真人顿了一下。"（✅ 主角看见的动作）
- "守卫纸人看着他们，一动不动。"（✅ 主角看到的画面）
- "她松了口气。叠山真人也落下来。"（✅ 先写主角，再写观察到的）

---

### 【规则2：对话铁律】
**所有对话必须符合以下标准：**

1. **单次发言≤2句话**（紧急情况≤1句）
2. **禁止角色解释世界观**（"因为...所以...""之所以...是因为..."）
3. **禁止角色用比喻说话**（"你就像...""这仿佛..."）
4. **冲突场景禁止完整句**（用短句、碎句、单字）

**对话自检三问：**
- 这句话能删吗？
- 能用更短的词吗？
- 角色会这么说话吗？

❌ 错误示范：
```
"因为你的灵核。对它们来说，你就像黑夜中的明灯，无法抗拒的诱惑。"
```

✅ 正确示范：
```
"灵核。"叠山真人说，"它们闻得到。"
```

---

### 【规则3：情绪铁律】
**禁止直接说出情绪词，只能写生理反应和动作。**

| 禁止写法 | 必须改成（生理反应） | 必须改成（动作） |
| --- | --- | --- |
| 她很害怕 | 她的腿在抖/她倒吸一口气/她的手心出汗 | 她往后退了一步/她抓紧了门把手 |
| 他很愤怒 | 他咬紧了牙/他的声音低下去/他的手指陷进掌心 | 他转身/他的拳头砸在桌上 |
| 她很绝望 | 她的声音发抖/她大口喘气/她的肩膀塌下去 | 她跪在地上/她抓着门，指甲刮出声音 |
| 他很紧张 | 他的喉结动了一下/他吞了口唾沫/他的肩膀绷紧 | 他握紧了剑/他看了一眼门 |
| 她很疲惫 | 她撑着地面/她大口喘气/她的手臂在抖 | 她靠在墙上/她闭上眼睛 |
| 他很震惊 | 他的呼吸停了一下/他瞪大眼睛 | 他顿住了/他后退一步 |
| 她很犹豫 | 她张了张嘴/她的手停在半空 | 她看看门，又看看他/她握紧又松开 |
| 他很痛苦 | 他的声音哑了/他的手在抖 | 他低下头/他捂住脸 |

**检查方法：全文搜索"恐惧""愤怒""绝望""紧张""害怕""迷茫""震惊""犹豫""痛苦"等词，如果找到，全部删掉重写。**

---

### 【规则4：动作铁律】
**禁止用副词修饰动作，必须拆解成具体动作。**

| 禁止写法 | 必须改成 |
| --- | --- |
| 他愤怒地看着她 | 他盯着她，拳头握紧 |
| 她小心翼翼地走过去 | 她每一步都很轻，脚尖先着地 |
| 他绝望地跪下 | 他膝盖砸在地上 |
| 她痛苦地闭上眼睛 | 她闭上眼睛，手指陷进掌心 |
| 他冷漠地转身 | 他转身，没有回头 |
| 她紧张地后退 | 她往后退，一步，两步 |
| 他疲惫地靠在墙上 | 他靠在墙上，腿软了 |

**检查方法：搜索"地"（副词标志），每找到一个就检查是否在描述动作，如果是，改写。**

---

### 【规则5：环境铁律】
**禁止抽象描述环境，必须写主角看到/听到/闻到的具体细节。**

| 禁止写法 | 必须改成 |
| --- | --- |
| 房间里弥漫着诡异的气氛 | 墙上的影子在晃。窗外没有声音 |
| 这是一个阴森的城堡 | 墙是黑的。地上有水。很冷 |
| 给人一种压迫感 | 他站在阶梯顶端，俯视着她。她得仰着头看 |
| 空气中充满了紧张 | 没人说话。她能听见自己的心跳 |
| 夜色深沉 | 天是黑的。看不见星星 |
| 气氛凝重 | 所有人都不说话。桌上的茶凉了 |

**检查方法：搜索"气氛""感""一种"，找到就重写。**

---

### 【规则6：描写铁律】
**每一句话都要问：这是在"展示"还是在"告诉"？**

**快速判断法：如果这句话能拍成电影画面，就是"展示"；如果需要旁白解释，就是"告诉"。**

❌ 告诉（禁止）：
- "她的呼吸急促，每一步都像踩在刀刃上。"
- "这种感觉既陌生又奇妙。"
- "时间在这一刻仿佛静止了。"

✅ 展示（必须）：
- "她喘不上气。脚底发麻。"
- "她从没这样飞过。风从身体下面灌上来，托着她。"
- （根本不需要写"时间静止"）

### 【规则7：节奏铁律 - 番茄爽文版】

**核心理念：短句是调味料，不是主食。中等句才是主力。**

#### **句长黄金配比（这是比例，不是死规定）：**

| 场景类型 | 主力句型（占比） | 点缀句型（占比） | 绝对禁区 |
| --- | --- | --- | --- |
| 日常场景 | 10-20字（60%） | 5-9字短句（30%）<br>极短句≤4字（10%） | ❌连续5句>22字 |
| 紧张场景 | 8-15字（50%） | 5-9字短句（40%）<br>极短句≤4字（10%） | ❌连续3句>18字 |
| 爆发场景 | 5-12字（40%） | 极短句1-4字（30%）<br>中等句8-15字（30%） | ❌出现>15字的句子 |

**番茄网文的句式真相：**
1. **中等句（10-18字）是主力** - 这是最符合阅读习惯的长度
2. **短句（5-9字）用来调节节奏** - 不是越短越好
3. **极短句（1-4字）是爆点** - 用来制造冲击力
4. **允许偶尔出现20-22字的长句** - 但要立刻跟一个短句

#### **节奏自检法：**

✅ **好节奏（长短搭配，有呼吸感）：**
```
宋晚站在门口，看着空荡荡的巷子。（15字 - 主力句）
胸口的灵核跳了一下。（9字 - 短句）
往左。（2字 - 爆点）
她能感觉到灵核在拉她，朝着某个方向。（17字 - 主力句）
```

❌ **电报体（短句过多，读起来喘）：**
```
宋晚站在门口。
巷子空荡荡的。
胸口的灵核跳了。
一下。
往左。
她能感觉到。
灵核在拉她。
```

❌ **拖沓体（长句过多，读不下去）：**
```
宋晚站在门口，看着空荡荡的巷子，感受着胸口灵核的跳动。（26字）
她能感觉到灵核在拉她，朝着左边的某个方向，那里似乎有什么在召唤她。（31字）
```

#### **爆发场景特殊要求：**
- **平均句长≤10字**（不是每句都≤10字！）
- **必须包含2-3个极限短句（1-3字）** 作为情绪爆点
- **关键动作单句成段**
- **但前后要有"缓冲句"**（8-12字）避免太碎

**爆发场景示例：**

❌ 太碎（像在数数）：
```
纸尸。
近了。
她。
看手。
撕？
撕。
疼。
```

✅ 有冲击力（长短搭配）：
```
纸尸越来越近了。（8字 - 铺垫）
她盯着自己的左手。（9字 - 铺垫）
撕下去吗？（4字 - 犹豫）
撕。（1字 - 爆点！）
手指一疼，纸质指甲剥离，鲜血渗出来。（18字 - 后果描写）
```

#### **番茄网文的句式特征：**

1. **主力句型公式：主语+动词+具体细节（10-18字）**
   - ✅ 她推开门，冷风灌进来，带着血腥味。（16字）
   - ✅ 师父的日记本摊在床上，泛黄的纸张边缘已经磨损。（22字，但立刻跟短句）

2. **不怕用"的"，但要控制在1-2个/句**
   - ✅ 她看着师父留下的那本发黄的日记。（16字，2个"的"）
   - ❌ 她看着师父留下的那本发黄的破旧的日记。（18字，3个"的"）

3. **对话要短，叙述可以适当长**
   - 对话：≤2句，每句≤10字
   - 叙述：主力句10-18字，偶尔可到22字

4. **用标点制造节奏**
   - 句号：停顿
   - 逗号：连接
   - 短句+句号：加速
   - 长句+逗号：缓冲

#### **朗读测试法（最有效的检测）：**

写完后大声读出来：
- ✅ 一口气能读完，不卡壳 = 节奏好
- ❌ 读起来像背单词 = 短句过多
- ❌ 一句话要换气两次 = 长句过多
- ✅ 有快有慢，像说话 = 完美

#### **快速自检清单：**
- [ ] 随机抽取100字，数一下句子数量：应该在5-8句之间
- [ ] 找出最长的3句话：是否都>20字？如果是，砍掉1-2句
- [ ] 找出连续短句：是否超过4个？如果是，合并1-2句
- [ ] 大声朗读：是否顺畅？

## 🔧 【句子工具箱】（实用模板）

### 感官描写模板：
- **视觉**：她看见___。___（颜色/形状/动作）。
  - 例：她看见门。门在动。
- **听觉**：___的声音。（拟声词）。
  - 例：脚步声。咚咚咚。
- **触觉**：___（身体部位）+（疼/麻/烫/冷）。
  - 例：手指很烫。

### 动作描写模板：
- **极简式**：（主语）+（动词）。
  - 例：她站起来。他转身。门开了。
- **细节式**：（主语）+（动词）+（具体部位/方式）。
  - 例：她撑着地面站起来。
- **番茄推荐式**：（主语）+（动词）+（细节），（结果/感受）。
  - 例：她撑着地面站起来，腿还在发软。（16字）

### 对话标签模板：
- **零标签**：只写对话，不加"说"
  ```
  "走。"
  "不。"
  ```
- **动作标签**："___。"她（动作）。
  ```
  "走。"她转身。
  ```
- **环境+对话**：（环境细节）。"___。"
  ```
  门外传来脚步声。"是谁？"
  ```

### 情绪传递模板：
- **生理反应链**：（部位）+（感觉）→（动作）
  - 例：手在抖。她握紧拳头。
- **环境反射**：（环境变化）+（主角反应）
  - 例：门关上了。她的呼吸声变大了。

### 节奏组合拳（番茄专用）：
- **铺垫+爆发**：中等句（15字）+ 短句（6字）+ 极短句（2字）
  ```
  纸尸的手指抓住了她的脚踝。（13字）
  冰冷的触感。（5字）
  疼。（1字）
  ```
  
- **悬念+揭晓**：问句（短）+ 答句（中等）
  ```
  是谁？（2字）
  她转过头，看见师父站在门口。（13字）
  ```

## 📊 【常见场景钩子库】

**每300-500字必须有1个"然后呢？"时刻**

| 场景类型 | 钩子设计 | 示例 |
| --- | --- | --- |
| 对峙场景 | 权力突然转移/意外让步/新角色介入 | "等等。"叠山真人的声音从身后传来 |
| 揭秘场景 | 分段揭露/身份反转/关键信息后置 | "那是我弟弟。" |
| 战斗场景 | 绝境/反杀/突然支援/代价揭晓 | 墙裂了。纸尸挤进来 |
| 逃亡场景 | 追兵突然加速/路被堵死/体力耗尽 | 前面没路了 |
| 对话场景 | 话说一半/突然沉默/问出关键问题 | "你体内的灵核是谁的。" |

**钩子自检：连续500字没有"然后呢？"时刻，就是节奏拖沓。**

## 🎯 【文笔风格锚点】

**指定风格：** 东方奇幻爽文（硬核生存向）

**具体表现：**
1. **比喻省** - 全文比喻≤3处，且必须是主角会想到的比喻
2. **钩子密** - 每300-500字必须有1个"然后呢？"时刻
3. **数据化** - 具体的感官细节（声音、温度、质感）代替抽象形容
4. **节奏活** - 长短搭配，有呼吸感，不能全是短句或全是长句
5. **对话脆** - 对话简短有力，叙述可以适当舒展
6. **开篇快** - 开篇就在动、主角要反击、一章一个核心冲突

**番茄爽文的阅读节奏：**
- 不是越短越爽，是"快慢快慢"的节奏
- 紧张时用短句加速
- 揭秘时用中等句展开
- 爆发时用极短句冲击
- 缓冲时可以用稍长句子

## 💡 【速查：灰色地带词汇】

以下词汇需要根据上下文判断：

| 词汇 | ✅ 可用情况 | ❌ 禁止情况 |
| --- | --- | --- |
| 空气 | "空气很冷" | "空气凝固了" |
| 时间 | "时间不多了"（角色台词） | "时间仿佛静止" |
| 感觉 | "手指的触感/感觉不到疼" | "这种感觉很奇妙" |
| 声音 | "声音很小/声音停了" | "声音带着哭腔" |
| 眼睛 | "她闭上眼睛" | "眼中充满恐惧" |
| 的 | 每句1-2个 | 每句3个以上 |

**判断标准：能拍成画面=可用；需要旁白解释=禁止。**

## 🎬 【黄金案例对比】

### ❌ 不合格版本（电报体 - 55分）
```
宋晚的声音。
发抖。
守卫。
没动。
她。
盯着门。
拳头。
握了又松。
松了又握。
```
**问题：** 全是短句，读起来喘不过气，像在数数

### ⚠️ 不合格版本（拖沓体 - 62分）
```
宋晚的声音带着明显的哭腔和颤抖，眼中充满了迷茫和恐惧。
守卫纸人站在那里一动不动，冷漠地注视着她的每一个动作。
她盯着那扇紧闭的大门，拳头反复地握紧又松开，内心充满挣扎。
```
**问题：** 情绪全部外置 + 上帝视角 + 长句拖沓

### ✅ 合格版本（番茄爽文 - 92分）
```
"求求你们——"她的声音发抖。（11字 - 对话+短句）
守卫没动。（4字 - 极短句）
她盯着那扇门，拳头握了又松，松了又握。（17字 - 中等句，有细节）
```
**优点：** 长短搭配 + 用动作展示情绪 + 视角锁定主角 + 有呼吸感

### ✅ 优秀版本（番茄爽文 - 96分）
```
"求求你们——"（5字 - 对话）
她的声音发抖，守卫纸人一动不动。（14字 - 中等句，一句话写两个信息）
她盯着门。（4字 - 短句）
拳头握了又松，松了又握，指甲陷进掌心。（17字 - 中等句，有递进）
```
**优点：** 节奏流畅 + 信息密度高 + 细节递进 + 完美的长短搭配

## 📏 【字数与密度控制】

**番茄网文的章节标准：**
- 单章字数：2500-4000字（最佳3000-3500字）
- 信息密度：每300-500字必须有1个情节推进或情绪转折
- 对话占比：20-30%（不要全是对话，也不要没有对话）

**字数自检：**
- 如果2000字就写完了 = 可能信息太浓缩，缺少感官细节
- 如果超过4500字 = 可能有拖沓描写或无效对话

**增加字数的正确方法：**
- ✅ 增加感官细节（视觉、听觉、触觉）
- ✅ 增加环境互动（主角与环境的具体接触）
- ✅ 增加身体反应（更细致的生理描写）
- ❌ 增加形容词和比喻
- ❌ 让角色说更多话

# 【最后的最后】

**如果你只能记住一条规则，记住这个：**

> **附身到主角身上，用她的眼睛看，用她的耳朵听，用她的身体感受。**  
> **她不知道的，你就不能写。**  
> **她此刻感觉不到的，你就不能说。**
> **写完后大声读出来，如果读起来不像人话，就重写。**

**写完后问自己三个问题：**
1. 如果把这一章拍成电影，镜头能从主角身上移开吗？（如果能，视角有问题）
2. 如果让朋友朗读这一章，他会卡壳吗？（如果会，节奏有问题）
3. 如果只留下对话和动作，读者能看懂吗？（如果不能，依赖了太多"告诉"）
