# 【第一部分：创作前看】（固定不变）

## ⚠️ 【致命错误清单】⚠️
**以下任何一条出现，本章直接判定为不合格：**

| 致命错误 | 具体表现 | 
| --- | --- |
| 🚫 作者解说腔 | "这种感觉很奇妙""时间仿佛静止了""他心中充满恐惧" |
| 🚫 上帝视角 | "两人都很紧张""他们不知道的是""与此同时，在另一边" |
| 🚫 情绪形容词 | "充满恐惧""带着哭腔""眼中的迷茫""心中的愤怒" |
| 🚫 过度比喻 | "像刀刃一样""仿佛黑夜中的明灯""如同萤火虫" |
| 🚫 角色讲课 | 角色说话超过2句连续对话/用比喻解释/说"因为...所以..." |
| 🚫 总结式心理 | "她意识到""她明白了""她感觉到一种前所未有的" |

---

## ✅ 【强制执行区】

### 【规则1：视角铁律】
**只能写主角此刻能感知到的内容。**

❌ 错误示范：
- "叠山真人心中一紧，他知道危险来了。"（你不是叠山真人）
- "守卫纸人冷漠地看着他们。"（"冷漠"是你的判断，不是主角看到的）
- "两人都松了口气。"（你只能写"她松了口气"）

✅ 正确示范：
- "叠山真人顿了一下。"（主角看见的动作）
- "守卫纸人看着他们，一动不动。"（主角看到的画面）
- "她松了口气。叠山真人也落下来，翅膀收起。"（先写主角，再写主角观察到的）

**检查方法：每写完一段，问自己"主角此刻能看到/听到/感觉到这个吗？"如果不能，删掉。**

---

### 【规则2：对话铁律】
**所有对话必须符合以下标准：**

1. **单次发言≤2句话**（紧急情况下≤1句）
2. **禁止角色解释世界观**（"因为...所以...""之所以...是因为..."）
3. **禁止角色用比喻说话**（"你就像...""这仿佛..."）
4. **冲突场景禁止完整句**（要用短句、碎句、单字）

❌ 错误示范：
```
"因为你的灵核。对它们来说，你就像黑夜中的明灯，无法抗拒的诱惑。"
```
← 太文艺了！叠山真人在逃命，不是在写诗！

✅ 正确示范：
```
"灵核。"叠山真人说，"它们闻得到。"
```

❌ 错误示范：
```
"站住！城堡已满，不接纳外来者！规则就是规则，城堡容量有限，不能超员。"
```
← 守卫话太多了！

✅ 正确示范：
```
"已满。"
左边那个说。
"不接纳。"右边那个重复。
```

---

### 【规则3：情绪铁律】
**禁止直接说出情绪词，只能写生理反应和动作。**

| 禁止写法 | 必须改成 |
| --- | --- |
| 她很害怕 | 她的腿在抖/她倒吸一口气/她往后退了一步 |
| 他很愤怒 | 他的手指陷进掌心/他咬紧了牙/他的声音低下去 |
| 她很绝望 | 她跪在地上/她的声音发抖/她抓着门，指甲刮出声音 |
| 他很紧张 | 他的喉结动了一下/他的肩膀绷紧/他吞了口唾沫 |
| 她很疲惫 | 她撑着地面/她大口喘气/她的手臂在抖 |

**检查方法：全文搜索"恐惧""愤怒""绝望""紧张""害怕""迷茫"等词，如果找到，全部删掉重写。**

---

### 【规则4：描写铁律】
**每一句话都要问：这是在"展示"还是在"告诉"？**

❌ 告诉（禁止）：
- "她的呼吸急促，每一步都像踩在刀刃上。"
- "这种感觉既陌生又奇妙。"
- "时间在这一刻仿佛静止了。"
- "给人一种无形的压迫感。"

✅ 展示（必须）：
- "她喘不上气。脚底发麻。"
- "她从没这样飞过。风从身体下面灌上来，托着她。"
- （根本不需要写"时间静止"）
- "他站在阶梯顶端，俯视着她。她得仰着头看。"

---

### 【规则5：节奏铁律】

**句子长度分配：**
- 日常场景：10-15字/句
- 紧张场景：5-8字/句  
- 爆发场景：1-4字/句

❌ 错误示范（爆发场景用长句）：
```
宋晚看着那些纸尸越来越近，心中涌起一股强烈的求生欲望，她知道自己必须做出选择。
```

✅ 正确示范：
```
纸尸更近了。
她看着自己的手。
撕。
```

**段落控制：**
- 每段≤4句话
- 爆发场景每段≤2句话
- 单句成段用于强调

---

## 🎯 【文笔风格锚点】

**指定风格：** 东方奇幻爽文（硬核生存向）

**具体表现：**
1. **比喻省** - 全文比喻≤3处，且必须是主角会想到的比喻
2. **钩子密** - 每300字必须有1个"然后呢？"时刻
3. **数据化** - 具体的感官细节（声音、温度、质感）代替抽象形容
4. **蹭热点** - 忌慢热（开篇就在动）、忌憋屈（主角要反击）、忌复杂（一章一个核心冲突）

## 🔧 【自检清单】（写完后必做）

### 第一遍：删除所有"作者腔"
- [ ] 搜索"仿佛""似乎""好像""宛如""犹如" → 能删就删
- [ ] 搜索"感觉到""意识到""明白了""知道了" → 改成直接写感官
- [ ] 搜索"心中""眼中""脸上" → 改成动作描写

### 第二遍：检查视角
- [ ] 每段问：这是主角看到的吗？主角此刻能知道这个吗？
- [ ] 搜索"两人""他们" → 改成"她...叠山真人..."

### 第三遍：检查对话
- [ ] 所有对话是否≤2句？
- [ ] 是否有角色在"解释""讲课""用比喻"？

### 第四遍：检查情绪
- [ ] 是否直接写了情绪词？
- [ ] 是否用生理反应和动作代替了？

### 第五遍：检查节奏
- [ ] 爆发场景的句子是否足够短？
- [ ] 是否有拖沓的描写？

## ⚠️ 【深度预处理提醒】⚠️

**创作流程（强制执行）：**

1. **第一步：读完所有信息（10分钟）**
   - 不要急着写
   - 在脑中过一遍完整剧情

2. **第二步：化身主角（5分钟）**
   - 闭眼，想象你就是宋晚
   - 你此刻在哪？
   - 你的身体什么感觉？
   - 你看到什么？听到什么？

3. **第三步：用主角视角过一遍关键场景（5分钟）**
   - 不要想"怎么写"
   - 只想"她会看到什么""她会做什么"

4. **第四步：开始写（快速输出）**
   - 不要边写边改
   - 先把整章写完

5. **第五步：用自检清单检查（必做）**
   - 至少过三遍
   - 每发现一个错误就记录下来

## 📊 【评分标准】

| 分数段 | 标准 |
| --- | --- |
| 90-100分 | 无致命错误 + 视角统一 + 对话精炼 + 情绪内化 + 节奏爆裂 |
| 80-89分 | 无致命错误 + 视角基本统一 + 对话可接受 + 情绪偶有外露 |
| 70-79分 | 1-2处致命错误 / 视角偶有混乱 / 对话略拖沓 |
| 60-69分 | 3-5处致命错误 / 上帝视角明显 / 过度解释 |
| 60分以下 | 致命错误遍布全文，需推倒重来 |

## 💡 【速查卡】

写作时遇到以下情况，立即停下来检查：

| 你写了什么 | 危险信号 |
| --- | --- |
| "她感到..." | 🚨 要改成具体感官 |
| "他心想..." | 🚨 要改成动作或直接写对话 |
| 对话超过3句 | 🚨 要拆分或删减 |
| "像...一样" | 🚨 要考虑是否必要 |
| "两人都..." | 🚨 要拆分成主角+观察到的另一人 |
| 用形容词描述情绪 | 🚨 要改成生理反应 |

## 🎬 【黄金案例对比】

### ❌ 不合格版本（68分）
```
宋晚的声音带着哭腔，眼中充满了迷茫和恐惧。
她心中升起一股愤怒和无力感。
时间在这一刻仿佛静止了。
```
**问题：**
- 情绪全部外置
- 上帝视角总结
- 比喻拖节奏

### ✅ 合格版本（92分）
```
"求求你们——"她的声音发抖。
守卫没动。
她盯着那扇门。拳头握了又松，松了又握。
```
**优点：**
- 用动作和对话展示情绪
- 视角锁定在主角
- 节奏快，信息密度高

# 【最后的最后】

**如果你只能记住一条规则，记住这个：**

> **附身到主角身上，用她的眼睛看，用她的耳朵听，用她的身体感受。**
