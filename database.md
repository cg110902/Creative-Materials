# 《纸人夜行》世界观数据库 
## 世界观
**时空**：现实世界与纸狱双层空间，可穿梭但受限，纸狱为主战场  
**场景**：(包括但不限于)装裱铺/纸扎店/纸狱白色荒原/纸城堡/青山镇/柳家废宅/古庙/纸质天空裂缝等  
**社会**：现实世界普通人类社会+纸灵隐秘生态，纸灵被视为禁忌，纸术传承濒临断绝，纸狱秩序崩坏  
**风格**：东方奇幻恐怖+纸文化美学，感官细节+纸质触感+折叠变形+撕裂音效，诡异氛围，避免套路化惊悚

## 核心主题
**主线**：揭开前世真相，封印纸狱主人→重建纸灵秩序

## 风格预设
**指定风格**：东方奇幻，女频向，诡异悬疑，纸文化美学，前世今生。

## 世界观补充细节

### 1. 纸灵生态体系

**纸灵分类：**
- **器物纸灵**：纸人/纸马/纸屋等祭品沾染怨念诞生
  - 初生期：刚获得意识，混沌懵懂
  - 觉醒期：具备完整思维，能言能行
  - 成熟期：掌握纸术，可自主行动
  - 堕落期：怨念反噬，化作纸妖

- **人化纸灵**：人类死后执念不散，灵魂附着纸物
  - 新郎新娘纸人(许舟/柳婉)
  - 面具纸人(许舟兄长)
  - 主持纸妖(柳芸)

- **天生纸灵**：纸狱中自然诞生，纯粹纸质生命
  - 守卫纸人：纸城堡守卫
  - 纸尸：堕落纸灵残骸
  - 折纸鹤(叠山真人)

**纸灵等级：**
- 下等纸灵：无意识，被动行动
- 中等纸灵：有意识，可交流
- 上等纸灵：掌握纸术，能战斗
- 纸妖：堕落强大，威胁人间
- 纸灵领主：统治纸狱一方(面具纸人)
- 纸道至尊：创世级存在(墨君)

**纸灵禁忌：**
- 不可主动伤害人类(除非正当防卫)
- 不可擅自离开纸狱进入现世
- 不可试图纸化活人
- 违反者被打散灵核或永久囚禁

---

### 2. 纸术体系

**基础纸术：**
- **纸遁**：将身体折叠变形，穿行缝隙或滑翔
  - 初阶：折叠四肢
  - 中阶：折叠全身成纸飞机
  - 高阶：瞬间折叠闪现

- **纸兵**：撕下身体纸片化作武器
  - 纸刀：锋利切割
  - 纸剑：刺穿攻击
  - 纸盾：防御阻挡
  - 消耗身体，需恢复

- **折纸术**：折叠空间或物体
  - 折叠空间：缩短距离
  - 折叠敌人：压缩束缚
  - 折叠法阵：布置陷阱

**进阶纸术：**
- **纸化**：将物体或生命纸质化
  - 物质纸化：石头金属纸化
  - 生命纸化：活物变纸(禁忌)
  - 空间纸化：区域纸化(柳芸/墨君级)

- **纸傀**：操控纸人作战
  - 单体操控：操控一个纸人
  - 群体操控：操控数十上百纸人
  - 纸人军团：千军万马(墨君级)

- **纸界**：构建纸质空间
  - 小型纸界：困敌小空间
  - 中型纸界：战场级别
  - 纸狱：世界级纸界(墨君创造)

**禁忌纸术：**
- **万物纸化**：将世界纸化(墨君专属)
- **灵魂纸囚**：囚禁灵魂于纸中
- **纸生**：用纸创造生命(逆天)

**纸郎特权：**
- 融合纸灵灵核获得其能力
- 在纸狱中身体不受限制
- 可感知纸灵情绪记忆
- 连接现世与纸狱的桥梁

### 3. 纸狱规则

**纸狱结构：**
- **白色荒原**：纸狱入口，空旷寂静
- **纸城堡**：纸灵聚居地，有秩序
- **纸尸区**：堕落纸灵游荡，危险
- **纸界深渊**：纸狱核心，封印墨君之地
- **现世裂缝**：连接现实的通道

**物理规则：**
- 身体变扁：进入纸狱后身体纸质化
- 重力减弱：可轻易跳跃滑翔
- 折叠空间：距离概念模糊
- 撕裂脆弱：身体易被撕但可恢复
- 灵核不灭：只要灵核在就能复活

**时间规则：**
- 纸狱时间流速：1天 = 现世1小时
- 停留过久：身体会完全纸化无法回归
- 临界点：7天(纸狱时间)

**能量规则：**
- 纸灵以怨念为食
- 纸狱能量来自现世祭品焚烧
- 焚烧过多：纸狱失衡，纸尸潮爆发
- 能量枯竭：纸狱崩溃，纸灵消散

### 4. 前世今生体系

**转世规则：**
- 怨念未消者：转世但会保留前世特征(相貌/胎记)
- 前世记忆：通常封印，特定触发才觉醒
- 触发条件：接触前世物品/遇到前世相关人/生死危机

**柳婉→宋晚转世线索：**
1. 相貌一模一样
2. 天生喜欢纸艺(前世爱好)
3. 莫名恐惧尖锐物(被刀刺死)
4. 梦中常见模糊场景(前世记忆)
5. 遇到许舟灵核后记忆觉醒

**前世关系网：**
- 柳婉(宋晚) ←→ 许舟：恋人，真心相爱
- 柳婉 ←→ 柳芸：表姐妹，柳芸嫉妒柳婉
- 许舟 ←→ 面具纸人：兄弟，深厚感情
- 柳婉 ←→ 老婆婆：柳家旧仆，知情人

**因果循环：**
- 前世冤案未破→今生重现
- 前世杀人凶手→今生纸妖
- 前世冤死情侣→今生再续前缘
- 前世创造纸狱→今生封印墨君

### 5. 核心冲突设定

**主要矛盾线：**

**纸灵 vs 人类矛盾**
- 人类视角：纸灵是诡异恐怖的怪物，应该消灭
- 纸灵视角：人类焚烧祭品创造了纸灵，却又抛弃迫害
- 冲突根源：纸灵由人类怨念创造，却被人类恐惧
- 和解可能：宋晚作为纸郎，连接两界，寻求共存

**秩序 vs 混沌矛盾**
- 秩序方：面具纸人维持纸狱规则，防止失控
- 混沌方：墨君要打破封印，纸化世界
- 冲突根源：墨君极端理念威胁两界平衡
- 高潮：宋晚等人联手封印墨君

**复仇 vs 放下矛盾**
- 复仇派：姜离要杀尽纸妖，白月瑶(若有)要报血仇
- 放下派：叠山真人劝导放下执念，许舟灵核选择守护而非报复
- 冲突根源：过去的伤痛如何面对
- 解决：仇恨的尽头是放下，但正义必须伸张

**前世 vs 今生矛盾**
- 宋晚身份混乱：我是宋晚还是柳婉？
- 情感纠葛：对许舟的感情是前世残留还是今生真心？
- 使命重压：必须完成柳婉未竟之事吗？
- 成长：接受前世，活出今生，融合为完整的自己
